# 2.7 String Methods
## Index Of
Index of returns the location of the requested element in a string

Counting starts at 0

`System.out.println(word.indexOf("o"));`

## Substring
Takes section of string

`System.out.println(word.substring(1));`

`System.out.println(word.substring(1,3));`

## Length
Prints length of string (-1 because starts at 0)

`System.out.println(word.length());`

## Equals and Compare To
Prints true or false if they equal or not

`System.out.println(word.equals(word2));`

word1 length - word2 length (this - other) | gets weird when different letters

`System.out.println(word.compareTo(word2));`
